# English Grammar Quiz — Level 2

Kuis grammar interaktif untuk siswa SMA (Kurikulum Merdeka), berdasarkan materi **English Grammar Level 2 — Penguasaan Verba & Struktur Kalimat** oleh Mr. Fio.

10 topik tingkat menengah, masing-masing dengan soal pilihan ganda, sistem **kunci jawaban** (lock-to-reveal), dan penjelasan dalam Bahasa Indonesia:

1. Tense Mastery
2. Modal Verbs
3. Conditionals
4. Passive Voice
5. Adjective Clause
6. Reported Speech
7. Gerund & Infinitive
8. Comparison
9. Causative
10. Question Tags

## Cara pakai

File ini **self-contained** — cukup buka `index.html` di browser. Tidak butuh server atau dependency.

### Publish ke GitHub Pages
1. Buat repo baru di GitHub, upload isi folder ini (`index.html` + `README.md`).
2. Repo → **Settings → Pages** → Source: `main` branch, folder `/ (root)`.
3. Tunggu beberapa menit, kuis live di `https://<username>.github.io/<repo>/`.

## Alur
Login (nama + kelas) → pilih topik → jawab tiap soal → **🔒 Lock Jawaban** untuk melihat penjelasan → hasil & nilai.
